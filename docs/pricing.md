# 模型价格

本站主要围绕 GPT API 的调用教程、接入方式和 API 中转方案展开。  
在实际使用 GPT API 时，最常见的两类计费方式是：

1. **文本模型按 Tokens 收费**
2. **图像模型按调用次数收费**

> API 平台地址：<https://jeniya.cn>

> 实时价格说明：<https://jeniya.cn/pricing>

## 计费原则

不同 GPT 模型在能力、速度、上下文长度和调用成本上存在差异，因此价格也会不同。一般来说：

- 能力更强的模型价格通常更高
- 轻量模型更适合高频调用和成本控制
- 图像模型通常按生成次数收费
- 平台中转价格可能与官方价格存在差异，应以实时页面为准

## 文本模型计费方式

文本类 GPT 模型通常按输入 Tokens 和输出 Tokens 分别计费。

常见适用模型包括：

- gpt-5.5
- gpt-5.4-mini
- gpt-5.4
- 其他 GPT 文本模型

**适合场景：**

- 聊天问答
- 文案生成
- 代码生成
- 知识库问答
- 智能客服
- 工作流自动化

## 图像模型计费方式

图像生成类 GPT 模型通常按调用次数收费。

常见模型包括：

- gpt-image-2

**适合场景：**

- AI 生图
- 文章配图
- 海报生成
- 电商图片辅助创作
- 视觉内容生产

## 常见计费问题

### Token 是怎么计算的？
文本模型通常按输入和输出分别计费。你发给模型的内容会消耗输入 Tokens，模型返回结果会消耗输出 Tokens。

### 为什么不同 GPT 模型价格差别很大？
因为不同模型的能力、速度、上下文长度、响应质量和资源成本不同，所以价格存在明显差异。

### 图像模型为什么按次数收费？
图像模型的成本通常按单次生成计算，因此更适合按调用次数收费，而不是按 Tokens 收费。

### API 中转价格为什么可能和官方不同？
不同平台的上游渠道、资源成本、运维成本和服务策略不同，因此中转价格不一定与官方完全一致。
